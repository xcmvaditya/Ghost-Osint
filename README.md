🔥 GHOST OSINT - Termux Installation Commands (Copy-Paste)

```bash

# GHOST OSINT TOOL - INSTALLATION COMMANDS
# ============================================

# Step 1: Update Termux
pkg update && pkg upgrade -y

# Step 2: Install Python & Required Packages
pkg install python git -y

# Step 3: Install requests library
pip install requests

# Step 4: Clone the repository
git clone https://github.com/xcmvaditya/Ghost-Osint.git

# Step 5: Enter the directory
cd Ghost-Osint

# Step 6: List files
ls

# Step 7: Give execute permission
chmod +x ghost_osint.py

python ghost_osint.py
```
